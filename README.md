У межах лабораторної роботи №1 з курсу «Розробка інтерфейсів користувача» було реалізовано застосунок “Форма опитування” у двох технологіях: Windows Forms та WPF. 
Програма поетапно відображає користувачу п’ять запитань щодо останньої прочитаної книги, дозволяє вводити відповіді у текстове поле, перевіряє коректність введення та зберігає результати у текстовий файл answers.txt. 
Після завершення опитування користувач отримує повідомлення про успішне завершення. 
У ході виконання роботи було закріплено навички створення графічного інтерфейсу, обробки подій та роботи з файловою системою в середовищі .NET.
